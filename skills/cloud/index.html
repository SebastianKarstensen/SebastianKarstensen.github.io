Cloud Infrastructure

Although I am a big fan of on-premise solutions and having full control of the hardware, I’ve naturally had to interact with the computing resources that various cloud providers offer. Through my work as a consultant, I have set up deployments of everything from small, singular Docker containers to multi-region VMs with custom autoscaling functionality.

Despite each cloud provider having its own web GUI and CLI tools, they essentially offer the same products hidden behind varying naming schemes. What persists across all platforms is a requirement to understand the underlying networking configuration and machine provisioning that makes “the cloud” such a powerful solution.

Platforms: Google Cloud, Microsoft Azure, AWS, Unity Gaming Services
